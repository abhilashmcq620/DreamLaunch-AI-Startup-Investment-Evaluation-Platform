🚀 DreamLaunch - AI Powered Startup Evaluation & Investor Decision Support Platform
Overview
DreamLaunch is an AI-powered startup evaluation and investor decision-support platform designed to bridge the gap between startup founders and investors. The platform leverages Large Language Models (LLMs) and Natural Language Processing (NLP) to transform unstructured startup pitches into actionable business insights, investment analyses, and due diligence reports.
The system provides separate dashboards for Innovators, Investors, and Administrators, enabling intelligent startup assessment, investor matching, risk analysis, and platform governance.

System Architecture
DreamLaunch follows a multi-role architecture consisting of:
1.	Innovator Dashboard
2.	Investor Dashboard
3.	Admin Governance Dashboard
4.	AI Processing Engine
5.	Analytics & Reporting Engine
6.	Database Management Layer
The platform processes startup pitches through 30 AI-powered analytical modules to generate structured insights and investment intelligence.

Installation
1.	Clone the repository
git clone https://github.com/abhilashmcq620/DreamLaunch-AI-Startup-Investment-Evaluation-Platform.git
2.	Navigate to project directory
cd DreamLaunch-AI-Startup-Evaluation
3.	Install dependencies
pip install -r requirements.txt
4.	Configure environment variables
Create a .env file and add:
GEMINI_API_KEY=your_api_key
SMTP_USER=your_email
SMTP_PASS=your_password
SECRET_KEY=your_secret_key
5.	Run the application
python app.py
